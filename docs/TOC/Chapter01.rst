第一章 基本命令
====================================

命令说明以简洁实用为主，不追求大而全。

随着 Linux 的发展，现在几乎所以的发行版都带有 GUI 图形桌面系统，桌面系统可以满足大部分的操作。但是在生产环境中几乎是没有图形界面的，大部分是通过 SSH 连接到服务器，从而依靠命令行操作服务器。Linux 的精髓也更多的体现在命令行上，其强大的功能，海量的工具，可以帮你轻而易举的完成各种复杂的系统管理操作。
