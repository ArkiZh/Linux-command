.. _cmd_touch:

touch 新建文件
####################################

touch 命令有两个功能：

1. 更新一个或多个文件的时间标签（文件内容不会改动）

2. 新建一个或多个空文件


命令格式：
************************************

.. highlight:: none

::

    touch [OPTION]... FILE...


常用选项：
************************************

::

    -a
        只更改存取时间

    -d, --date=STRING
        使用指定的日期时间，而非现在的时间

    -m
        只更该变动时间

    -r, --reference=FILE
        按指定文件的时间更新时间


使用实例：
***********************

::

    [Linux]$ ls
    aaa  bbb
    [Linux]$ touch ccc
    [Linux]$ ls
    aaa  bbb  ccc
